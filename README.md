# URLShortener API

Сервіс для скорочення довгих посилань з підтримкою авторизації та аналітики.

## Технології
- **Java 21 / Spring Boot 3.4.2**
- **PostgreSQL 18** (з міграціями Flyway)
- **Spring Security + JWT**
- **JUnit 5 / Mockito** (Unit-тести)
- **Docker / Docker Compose**

##  Налаштування
Перед запуском створіть файл `.env` у корені проєкту та задайте змінні для бази даних:
- `DB_USERNAME=postgres`
- `DB_PASSWORD=ваш_пароль`

## Як запустити
1. Зберіть проєкт: `./gradlew build`
2. Запустіть сервіс: `docker-compose up --build`
   *Додаток буде доступний на порту **8082**.*

## Документація (Swagger)
Після запуску Swagger UI доступний за адресою:
`http://localhost:8082/swagger-ui/index.html`
*Для доступу до захищених методів використовуйте кнопку **Authorize** та вставте JWT токен (отриманий після /auth/login).*

## Архітектура (Package-as-feature)
Проєкт організований за функціональними модулями:
- `ua.opnu.url_shortener.auth` — реєстрація користувачів та JWT авторизація.
- `ua.opnu.url_shortener.link` — генерація скорочених кодів (8 символів), редіректи та керування посиланнями.
- `ua.opnu.url_shortener.stats` — аналітика переходів (топ-3 посилання та статистика за сьогодні).
